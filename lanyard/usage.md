## Usage

Join the Lanyard [Discord](https://discord.com/invite/WScAm7vNGF) for Lanyard Profile ReadME.

In a `README.md` file, include the following, replacing `:id` with your Discord user ID: \
Help needed! I don't know where can I find my User / Server / Message ID? [Click here!](https://support.discord.com/hc/en-us/articles/206346498-Where-can-I-find-my-User-Server-Message-ID)

```md
[![Discord Presence](https://lanyard.cnrad.dev/api/:id)](https://discord.com/users/:id)
```

It should display something similar to the following:

[![Discord Presence](https://lanyard.cnrad.dev/api/408005465423872001)](https://discord.com/users/408005465423872001)
